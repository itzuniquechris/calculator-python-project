# ⚡ Appliance Cost Calculator

A simple and effective Python tool to estimate the monthly electricity cost of any home appliance. 

## 🚀 Features
- **Cost Calculation:** Converts Watts and daily usage into a monthly USD total.
- **Safety Buffer:** Uses Python's `math` module to provide a "rounded up" estimate for budgeting.
- **Smart Alerts:** 
  - 🚩 Flags high energy usage (over $50).
  - 🌱 Awards an "Eco-Discount" for low usage (under $5).

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Modules:** `math`

## 📖 How to Use
1. Run the script: `python appliance_cost_calc.py`
2. Enter the appliance name (e.g., Air Conditioner).
3. Input the wattage, hours used per day, and your local price per kWh.
4. View your monthly breakdown and savings status!
